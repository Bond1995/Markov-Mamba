# Markov-Mamba
Code to experiment with SSM models (Mamba) using data generated from k-th order Markov sources.
The model architecture is a very essential (but complete) implementation of the original Mamba2 architecture, based on the code in the official [Mamba repository](https://github.com/state-spaces/mamba/), Copyright (c) 2023, Albert Gu, Tri Dao.
